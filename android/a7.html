どうだったかな


どうだったか？
思い出せるか？をやってみる記憶力を試すアプリです。

スタートを選択すると
(1)簡易モード
4枚の画像が表示されます。
(2)ノーマルモード
9枚の画像が表示されます。
(3)ハードモード
16枚の画像が表示されます。

表示された画像を覚えてください。
どんな画像があったのか？
を覚えてから
次へボタンを選択してください。

問題として4個のボタンが表示されます。
問題の種類として、下記の種類の問題が表示されます。
(1)あった画像は？
(2)なかった画像は？
という問題の指示と
覚えていた9枚の画像から
4個の選択肢のボタンの中から適切な答えを選択してください。

間違えてしまうと、正解のボタンの背景がピンク色になります。
もう一度ボタンを選択すると
リセットされます。

正解すると正解数がカウントアップされます。

戻るボタンを選択するとスタート画面に戻ります。

言語切替について
言語切替はスタート画面で選択できます。

ヘルプについて
このアプリの使い方を説明しています。
参考になれば、幸いです。

by nekofes.github.io

What was that?


How was it?
Can you remember? This is an app that tests your memory.

If you select simple mode
Four images will be displayed.
Remember the four images.

When you select start
(1) Easy mode
4 images will be displayed.
(2) Normal mode
9 images will be displayed.
(3)Hard mode
16 images will be displayed.

What kind of images were there?
After remembering
Please select the Next button.

Four buttons are displayed as a problem.vThe following types of problems are displayed as problem types:
(1) What was the image?
(2)What images were missing?
Instructions for the problem
From the 4/9/16 images you remembered (from 4 images in simple mode)
Choose the appropriate answer from the four choice buttons.

If you make a mistake, the background of the correct button will turn pink.
Select the button again
It will be reset.

If you answer correctly, the number of correct answers will be counted up.
