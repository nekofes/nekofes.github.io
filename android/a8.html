けんけんぱ


このアプリについて
けんけんぱして女の子を動かしてね。
左足
両足
右足
のいずれかを右側の背景ピンク色の足の位置と同じ操作ボタンを選んでね。
右上にもテキストメッセージで
ユーザから見たいずれかのけんけんぱを表示しています。
間違うとゲームオーバーです。

by nekofes.github.io

ゲーム開始

1回目
画面右側の一番下の両足がピンク色になっています。
これが現在の足の位置です。
この足の位置と同じ両足の状態を下の3個並んでいる両足の状態から選んでください。
右側の足の状態は、一番下のピンク色が今回の両足の位置
その上が次の両足の状態となります。
上の位置がこの先の両足の状態を示すので
この足の状態を見ながら、どんどん両足の状態を選択してください。


2回目
1回目の考え方と同じように右側のピンク色の両足の状態と同じものを下の両足の中から選んでください。


ヘルプ
使い方が記載されています。




おしらせ
楽しんでもらえればなによりです。
はやくやっても間違えないようになってください。

Game Start

1st time
The feet at the bottom right of the screen are colored pink.
This is the current position of the feet.
Select the same foot state as this foot position from the three foot states lined up below.
For the foot state on the right, the bottom pink is the current position of the feet
The state above that is the next state of the feet.
The position above indicates the future state of the feet, so
While looking at this foot state, select the state of the feet one after another.


Second time
Just like the first time, please choose the pair of legs below that matches the condition of the pink legs on the right.


HELP
Instructions on how to use it are included.




Announcement
I hope you enjoy it.
Please try to do it quickly so that you don't make any mistakes.
