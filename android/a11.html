ゲームの例
「この絵は何番目？」の初級サンプルです。
画面(1番目)

初級では3枚の画像が用意されています。
最初の画面なので「1番目」と表示されています。
次のボタン「＞」をタップすると2番目の画像が表示されます。
画面(2番目)

次のボタン「＞」をタップすると3番目の画像が表示されます。
画面(3番目)

最後の画像が表示されると問題ボタンが表示されます。
問題ボタンをタップすると問題画面が表示されます。
前の画像を確認したい場合は前へボタンの「＜」ボタンをタップしてください。
問題の画面

この例では、2番目の画像を選ぶと正解です。
スコアがインクリメントされます。
次へボタンを選択すると、次の問題が表示されます。
間違うとゲームオーバーです。
もう一度ボタンが表示されるので、選択するとスコアがリセットされ、ゲームが再開されます。

Game Examples
This is a beginner's sample of "What number is this picture?"
Screen (1st)
What number is this image?
There are three images available for easy mode.
Since this is the first screen, it says "1st."
When you tap the next button ">," the second image will be displayed.
Screen (2nd)

Tap the next button ">" to display the third image.
Screen (3rd)

When the last image is displayed, the Problem button will appear.
When you tap the Problem button, the Problem screen will be displayed.
If you want to see the previous image, tap the previous button "<".
Problem screen

In this example, if you select the second image, the answer is correct.
Your score will be incremented.
If you select the Next button, the next problem will be displayed.
If you get it wrong, the game is over.
The button will be displayed again, and if you choose it, your score will be reset and the game will start again.
