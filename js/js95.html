現在開いているページの上端のy座標と左端のx座標
window.scrollYとwindow.scrollXプロパティ

現在開いているページのスクロール位置を取得する場合は
window.scrollYとwindow.scrollXプロパティを使用します。



[サンプル]
copy
const btn = document.getElementById("btn");
btn.addEventListener("click", () => 
{
	scrollPositionXY();
});
function scrollPositionXY()
{
	const position = positionXY();
	console.log("x:" + position.x + " y:" +position.y);
}
function positionXY()
{
	return {
		y:window.scrollY,
		x:window.scrollX
	};
}



ボタンをクリックイベントで選択するとscrollPositionXY()関数を実行します。

return {
y:window.scrollY,
x:window.scrollX
};
positionXY()関数ではwindow.scrollYとwindow.scrollXプロパティを使用して
x座標とy座標の戻り値を返します。
