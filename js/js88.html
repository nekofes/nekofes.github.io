join
Array.prototype.join()

配列のすべての要素を連結し、1つの文字列を返します。

[サンプル]
copy
const aryTest = [1, 2, 3];
console.log(aryTest.join());//1,2,3
console.log(aryTest.join(""));//123
console.log(aryTest.join("-"));//1-2-3
console.log(aryTest.join("	"));//1	2	3
console.log(aryTest.join("\n"));
/*
1
2
3
*/

aryTest.join(結合する文字列)
空文字の場合は、配列の要素をそのまま結合します。
空文字以外は、結合する文字列を用いて要素と要素を結びつけます。
ただし、未指定の場合はデフォルトである「,」(カンマ)で結合されます。
配列の要素が文字列に変換され、連結されて返されます。
連結された文字列は新しい文字列であり、元の配列は変更されません。
配列に1つしか要素がない場合、区切り文字は使用されません。
空の配列でjoin()を呼び出すと、空の文字列が返されます。
console.log(aryTest.join("));//3
上記は、タブ文字で処理しています。
