JSONの配列変換
JSON形式の文字列データの変換

javascript組み込みのJSONオブジェクトの
stringify()
parse()
を使用して変換および復号できます。

[サンプル]
copy
//[test 1]
const ary1 = [1, 2, 3];
const string1 = JSON.stringify(ary1);
localStorage.setItem("ary1", string1);
const string2 = localStorage.getItem("ary1");
const result2 =JSON.parse(string2);
console.log(result2);//[1, 2, 3]

//[test 2]
const ary2 = {animal:"panda", food:"bamboo"};
const string3 = JSON.stringify(ary2);
console.log(string3);
localStorage.setItem("ary2", string3);
const string4 = localStorage.getItem("ary2");
const result3 =JSON.parse(string4);
console.log(result3);//{"animal":"panda","food":"bamboo"}

[test 1]
const string1 = JSON.stringify(ary1);
JSON.stringifyを使用して配列をJSON形式の文字列に変換します。

localStorage.setItem("ary1", string1);
localStorageにary1という名前で保存します。

const string2 = localStorage.getItem("ary1");
JSON文字列を取得します。

const result2 =JSON.parse(string2);
JSON.parseで元の配列に戻します。


[test 2]
連想配列のようなクラス配列も同様に変換および復号できます。
